🔐 Página de Login e Cadastro

Este projeto apresenta uma interface de autenticação com páginas de Login e Cadastro, desenvolvida utilizando HTML5, CSS3 e JavaScript.

O objetivo é demonstrar a criação de formulários interativos, manipulação do DOM e construção de layouts modernos e responsivos no front-end.

📌 Objetivo do Projeto

Este projeto foi desenvolvido para praticar e demonstrar conhecimentos em desenvolvimento front-end, incluindo:

Estruturação semântica com HTML5

Estilização moderna e responsiva com CSS3

Manipulação do DOM com JavaScript

Implementação de formulários de login e cadastro

Criação de interfaces interativas para o usuário

🛠️ Tecnologias Utilizadas

HTML5

CSS3

JavaScript

✨ Funcionalidades

O projeto inclui as seguintes funcionalidades:

Formulário de Login

Formulário de Cadastro

Alternância dinâmica entre login e cadastro utilizando JavaScript

Validação básica de campos

Interface moderna e intuitiva

Layout responsivo

📱 Responsividade

A interface foi desenvolvida para se adaptar a diferentes dispositivos:

💻 Desktop

📱 Smartphones

📲 Tablets
