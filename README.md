# 🔐 Página de Login e Cadastro

Interface de autenticação com páginas de Login e Cadastro, desenvolvida com HTML5, CSS3 e JavaScript. O projeto demonstra a criação de formulários interativos, manipulação do DOM e construção de layouts modernos e responsivos no front-end.

🔗 **Demo:** [franciellyvicente.github.io/cadastro-login](https://franciellyvicente.github.io/cadastro-login/)

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript

## ✨ Funcionalidades

- Formulário de Login
- Formulário de Cadastro
- Alternância dinâmica entre login e cadastro utilizando JavaScript
- Validação básica de campos
- Interface moderna e intuitiva
- Layout responsivo (desktop, smartphones e tablets)

## 🎨 Design

- Fundo com sombreado em tons de preto e cinza, mais escuro nas bordas
- Botões de login e cadastro coloridos, com animação de hover ao passar o mouse
- Transição de cor ao alternar entre as páginas de login e cadastro

## 📌 Objetivo do Projeto

Este projeto foi desenvolvido para praticar e demonstrar conhecimentos em desenvolvimento front-end, incluindo:

- Estruturação semântica com HTML5
- Estilização moderna e responsiva com CSS3
- Manipulação do DOM com JavaScript
- Implementação de formulários de login e cadastro
- Criação de interfaces interativas para o usuário

## 📌 Observações

- Este projeto possui validação básica no front-end
- Não possui integração com back-end ou banco de dados
- Desenvolvido com fins educacionais e prática de front-end
